PRAXIS product roadmap

Что осталось сделать

Это рабочий чеклист после текущего состояния MVP. Сейчас у нас есть local full-stack prototype: frontend, backend, JSON database, AI Intake endpoint, optional LLM adapter, Eval Runner, Pilot Runs, Knowledge Base, Retrieval, OpenAPI import, MCP scaffold, Governance checks, backend Agent Runtime и Human Handoff Queue. Дальше нужно превращать это в enterprise-grade FDE Operating System.
